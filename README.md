# College ERP Software

A web-based College ERP software designed to manage students, faculty, academics, attendance, examinations, and administrative operations efficiently.

## Features
- Student Management
- Faculty Management
- Faculty & Staff Dashboard
- Subject & Course Management
- Attendance Tracking
- Examination & Result Management
- Role-based Access (Admin / Staff)

## Tech Stack
- Python
- Django
- SQLite
- HTML, CSS, JavaScript, Bootstrap

## Installation
1. Clone the repository  
   `git clone https://github.com/sthuthidolpady/College--ERP-Software.git`

2. Install dependencies  
   `pip install -r requirements.txt`

3. Run migrations  
   `python manage.py migrate`

4. Start the server  
   `python manage.py runserver`
